# Xclovis
Projeto de teste para Xcloud no ubuntu minimal 18.0.4

## Git Clone
```bash
sudo apt-get install git
git clone https://github.com/LucasBello/Xclovis.git
```
## Permissionar os sh
```bash
cd Xclovis
sudo chmod +x xclovis.sh
sudo chmod +x install.sh
```

## Instalar
```bash
./install.sh
```
## Rodar
./xclovis.sh
```
